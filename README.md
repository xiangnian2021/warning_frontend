# 基于Vue的Cron表达式组件

#### Description
基于Vue的Cron表达式组件,封装比较简单，没做组件优化，供大家参考


#### Installation

1. 下载本项目
2. 安装依赖包
  npm install
3. 运行
  npm run dev
